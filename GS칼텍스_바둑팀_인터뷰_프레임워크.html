핵심① 1지명 재선발 기준 — 42.9% 승률 실패 원인 진단 ▾
원성진 9단이 선발 당시 랭킹 8위에서 시즌 중 25위로 하락했는데, 선발 전 기량 하락세를 사전에 감지하지 못한 구조적 원인이 무엇이라고 보나요?
→ 내부 문서에서 "선발 당시 랭킹 8위 → 현재 25위"로 하락세를 간과한 것이 명시됨. 감독의 정상급 선수 경기력 실시간 확인 시스템 부재가 문제
↳ 후속: "1지명 선발 시 단순 랭킹 외에 어떤 지표를 봐야 하락세를 조기 포착할 수 있었을까요?"
리그 상위권 팀에서 1지명이 6~7할 승률을 유지하는 반면, GS의 1지명이 4할대로 떨어진 근본적 차이가 무엇인가요?
→ 내부 분석에 따르면 "1지명자 2:3 패배 경기에서 2승 5패" — 에이스가 결정국에서 특히 약했음
26-27 드래프트에서 1지명 후보군으로 주목해야 할 선수가 누구인가요? 연봉제 전환 검토 시 영입 가능한 선수는요?
→ 바둑리그 선수 선발 방식이 추첨 → 1선발 연봉제로 변경 검토 중. 이 변화에 선제 대응 필요
내부 데이터
1지명이 5할 이하 승률일 경우 5국에서 패해 2:3 패 가능성이 급증. 이는 단순히 "에이스가 약했다"가 아니라 오더 구조 전체의 문제임 — 1지명 부진을 다른 지명자가 보완할 수 있는 오더 설계가 없었음.
핵심② 4~5지명 즉각 교체 — 하위 지명 선발 프로세스 개선 ▾
권효진 7단이 12.5% 승률(1승 7패)을 기록했는데, 선발 전 이 결과를 예측할 수 있었던 신호가 있었나요? 4지명 선발 시 어떤 평가 기준이 보완되어야 했을까요?
→ 내부 문서는 "유망주 및 성장 가능성 판단 실패, 선수 선발 시 경기력 평가 세분화 필요"를 명시. 선발 프로세스 자체가 문제
리그 내 4~5지명에서 45% 승률을 안정적으로 유지하는 팀들은 어떤 기준으로 하위 지명 선수를 선발하나요? 전투형/실리형 등 스타일 구분이 실제로 유효한가요?
→ 26-27 계획에서 "로테이션 Pool 운영(전투형/실리형)"이 언급됨. 이 전략의 현실적 효과를 검증
26-27 목표
4~5지명 승률 45% 목표 설정. 현재 권효진 12.5%, 김승재 33.3%에서 45%로 올리려면 선수 교체 없이는 불가능. 전문가에게 "이 목표가 현실적인가" 직접 확인 필요.
추가① 투샤오위 잔류 가치 & 외국인 선수 전략 ▾
투샤오위가 60% 승률(3승 2패)로 선방했는데, 2년차 시즌에서 리그 적응도가 더 오를 것으로 보나요? 잔류 vs 신규 영입, 어떤 선택이 더 유리한가요?
→ 내부 계획에서 "투샤오위 잔류 또는 동급 이상 외국인 선수 선발" 검토 중. 리그 경험 1년차 vs 신규 영입의 트레이드오프를 전문가 시각으로 평가
경쟁팀(합천 판인 2년차, 원익 진위청 2년차, 영림 당이페이 3년차)의 외국인 선수 활용 방식과 비교해서, GS의 외국인 선수 전략적 배치(상대 에이스 맞대결 또는 하위 지명자 상대 1승 확보)가 어떻게 설계되어야 할까요?
→ 26-27 계획에서 "중국 선수 전략 카드화"가 명시됨. 구체적 배치 전략 논의

▾
핵심① 14경기 중 7회 2:3 석패 — 결정국 패패 패턴 분석 ▾
14경기 중 7경기가 2:3 석패였는데, 이 패턴이 "선수 역량의 문제"인가, "오더/전략의 문제"인가, 아니면 "결정국 심리 관리의 문제"인가요?
→ 내부 문서는 "전력의 붕괴가 아닌 구조적인 실패. 접전에서 매번 지는 팀"이라고 진단. 이 구조적 실패의 정확한 원인 규명 필요
↳ 후속: "3:2로 이긴 2회와 2:3으로 진 7회의 차이에서 발견되는 공통점이 있나요?"
결정국(5국)에서 GS팀이 반복적으로 지는 패턴이 있다면, 그게 포석 단계 문제인가요, 중반 전투인가요, 아니면 끝내기/시간 관리 문제인가요?
→ 피셔룰 하에서 결정국은 심리 압박이 가장 크고 시간 관리 실패가 치명적. 구체적 패인 분석 요청
핵심 수치
3:2 승리 단 2회 vs 2:3 패배 7회 — 이 비율 차이만으로도 "팀 전력이 아닌 접전 운영 능력"이 핵심 문제임을 보여줌. 전문가에게 이 수치를 제시하고 리그 내 유사 패턴 팀 사례와 비교 요청.
핵심② 오더 전략 재설계 — 상대팀에 읽힌 기용 패턴 탈피 ▾
내부 분석에서 "상대팀이 GS칼텍스 기용 패턴에 적응했다"고 했는데, 구체적으로 어떤 패턴이 읽혔다고 보나요? 변칙 오더를 효과적으로 사용하는 팀의 사례가 있나요?
→ 26-27 계획에서 "오더 전략 재설계: 2~3지명 중심 승점 확보 대응, 1지명 변칙 포함" 검토 중. 리그 내 성공 사례 확인 필요
정관장(8위), 수려한 합천(5위), 한옥마을 전주(3위)에게 2전 2패를 기록했는데, 이 팀들에 대한 공통 약점이 있나요? 상대별 맞춤 오더 전략 수립이 현실적으로 가능한가요?
→ 영암에는 2전 2승이지만 특정 팀에 편중된 패배 패턴이 있음. 데이터 기반 상대별 전략 수립 가능성 확인
추가① 포스트시즌 진출 목표 — 현실성과 필요 조건 ▾
26-27 시즌 포스트시즌 진출 목표가 현실적으로 가능하려면 최소한 어떤 조건이 갖춰져야 하나요? 1지명 60% + 4~5지명 45% 목표가 달성된다면 순위가 어느 수준이 될까요?
→ 영림창호팀 해체 확실시로 26-27 리그는 7팀 체제가 될 수 있음. 팀 수 감소가 포스트시즌 진출에 유리하게 작용할 수 있음

핵심① 감독 교체 기준 — 14년 체제 vs 3연속 포스트 탈락 ▾
바둑리그에서 감독 교체가 팀 성적에 실질적인 영향을 미치는 사례가 있나요? 교체 이후 성적이 반등한 팀과 오히려 악화된 팀의 공통점은 무엇인가요?
→ 내부 문서에서 "22년간 총 37개 팀 창단/해체" — 감독 교체 사례 자체가 드묾. 전문가에게 교체 효과를 객관적으로 평가 요청
↳ 후속: "GS의 경우 3연속 포스트 탈락과 친소관계 선발 문제가 있는데, 이 정도면 교체가 정당화되는 수준인가요?"
차기 감독이 갖춰야 할 최우선 역량이 무엇인가요? "정상급 선수 기량 실시간 파악", "선수단 조직 장악력", "오더 전략 수립", "중국 선수 네트워크" 중 어느 것이 가장 중요한가요?
→ 내부 문서의 감독 역할 기준(선수 선발·오더 작성, 선수단 관리, 의사결정력, 인터뷰 능력)을 외부 전문가 시각으로 검증
내부 문서 기준
감독 후보군으로 목진석(국가대표 감독 역임, 중국어 능통), 박정상(영림팀 우승, 대중 인지도 높음, 중국 네트워크), 김지석(Mr. GS칼텍스, 현역 상위 랭커), 한종진(리그 감독 다수 경험, 여수 출신) 등 8인이 검토됨.
핵심② AI 기보 분석 & 데이터 기반 선수 선발 체계 ▾
선수 선발 시 AI 기보 분석을 활용해 "랭킹 외 하락세·스타일 적합성"을 파악하는 팀이 있나요? 이 시스템을 도입한다면 어떻게 구현하는 게 현실적인가요?
→ 내부 분석에서 "선수 선발 시 경기력 평가 세분화 필요"가 명시됨. AI를 활용한 사전 스크리닝 시스템 구축 가능성 확인
상대팀 기용 패턴을 사전 분석해서 오더에 반영하는 체계적 프로세스를 갖춘 팀이 리그 내에 있나요? 그 팀의 방법론을 어떻게 적용할 수 있을까요?
→ 결정국 패배 개선을 위한 데이터 기반 오더 전략 수립 가능성 탐색
추가① 피셔룰 속기 대응 & 선수 컨디션 관리 ▾
25-26 시즌에서 GS 선수단이 피셔룰 속기에서 시간 압박 관련 실수가 있었나요? 경쟁팀 대비 속기 적응 수준은 어떻게 평가하나요?
